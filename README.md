# AutoOutreach-AI
An automated outreach pipeline that sends **context-aware, personalized emails** to professors and hiring managers using a **live Google Sheet as the control plane**.
This system knows *me* (skills, projects, research, resume variants), knows *them* (role, lab/company, intent), and decides **what email to generate and which resume to attach**—without manual micromanagement.



 ## Core Idea

- Google Sheet = single source of truth  
- LLM = reasoning engine  
- Email service = delivery  
- Resume selector = strategy layer  

Update the sheet → pipeline adapts automatically.

---

## Features

- Personalized email generation (professors / recruiters)
- Resume selection based on role, domain, and seniority
- Dynamic subject lines and tone control
- Sheet-driven workflow (add, pause, retry, track status)
- Logging of sent emails + outcomes
- Safe-guards against duplicate or low-quality outreach
